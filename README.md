# CMP Reimagined — Case Study

A case study on redesigning JobHai's (Naukri.com / InfoEdge) recruiter-side Candidate Management Page — the single screen recruiters use to filter, unlock, and contact candidates.

**Live:** https://anandparth.github.io/CMP-Case-Study/

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- Framer Motion

## Local dev

```bash
npm install
npm run dev
```

`npm run build` outputs a static build to `docs/`, served directly via GitHub Pages (Settings → Pages → branch `main` / folder `/docs`).

## Notes

Screens shown are pulled from the working Figma file. Candidate names and details are placeholder data used for design work, not real users. This is an independent case study reflecting the author's own process and perspective, not an official JobHai / InfoEdge publication.
